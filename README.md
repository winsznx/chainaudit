# ChainAudit

Immutable audit trail system for compliance and verification on Base and Stacks blockchains.

## Features

- Record audit entries with cryptographic proof
- Multi-auditor support
- Timestamped records
- Verify data integrity
- Complete audit history

## Tech Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Base**: Solidity 0.8.20, Foundry, Reown wallet
- **Stacks**: Clarity v4, Clarinet, @stacks/connect

## Getting Started

```bash
pnpm install
pnpm dev
```
